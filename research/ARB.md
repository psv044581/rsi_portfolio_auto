## ЧТО ЭТО

**Arbitrum** — ведущий Ethereum L2 на базе оптимистичного роллапа. Разработан **Offchain Labs** (Принстон, 2019), mainnet запущен **31.08.2021**. Протокол выполняет транзакции вне Ethereum с финализацией через fraud proofs (BOLD), сохраняя безопасность основной сети.

**Для кого:** DeFi-протоколы (GMX, Uniswap V3, Radiant, Gains Network), институциональные клиенты (кастомные Orbit-цепи), разработчики (Stylus — смарт-контракты на Rust/C/C++).

**Реальные пользователи:** ✅ Да — 1.45M активных кошельков (фев. 2026), 2.1B+ обработанных транзакций, 34% доля L2-рынка по транзакциям. TVL сети $1.41B, Orbit-экосистема $16.63B.

## ТЕХНОЛОГИЯ

| Компонент | Описание | Зрелость |
|---|---|---|
| **Arbitrum Nitro** | Core stack: AVM + WASM, гарантии EVM-совместимости | 🟢 Production |
| **BOLD (Bounded Liquidness Delay)** | Новые fraud proofs с ограниченным временем оспаривания (~7 дней) | 🟡 В деплое |
| **Stylus** | WASM смарт-контракты на Rust/C/C++ (10–100x дешевле Gas) | 🟡 Launched 2024 |
| **Orbit (L3 chains)** | Фреймворк кастомных цепей (Xai Games, Robinhood chain) | 🟢 Production |
| **Timeboost** | Приоритизация транзакций за плату (аукцион) — новый revenue stream | 🟢 Live с 17.04.2025 |
| **AnyTrust** | Облегчённый вариант для приложений (Arbitrum Nova) | 🟢 Production |

**Ключевые инновации:** Nitro — первый L2 с WASM-компилятором EVM. BOLD устраняет vector of multi-party delay attacks. Orbit превращает Arbitrum в L2-as-a-Service платформу.

**Аудиты безопасности:** Trail of Bits, Certik, Offchain Labs (внутренние). Security Council — 12 multisig-хранителей для emergency actions (новый состав с 21.05.2026).

**Инцидент:** KelpDAO hack 18.04.2026 (~$293M в LayerZero bridge) — не уязвимость Arbitrum, но Security Council заморозил $71M от хакеров; SDNY суд выдал ордер на конфискацию (связь с Lazarus Group).
