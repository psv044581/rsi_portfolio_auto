## ЧТО ЭТО

**Ethereum (ETH)** — глобальная программируемая блокчейн-платформа для смарт-контрактов и децентрализованных приложений, запущенная **30 июля 2015 года**. Создана Виталиком Бутериным с командой из 8 сооснователей после публикации белпэйпера в ноябре 2013 г.

**Для кого:** Разработчики (деплой dApps), участники DeFi, NFT, GameFi, институциональные инвесторы (Spot ETF), корпоративные клиенты.

**Реальные пользователи:**
- 400M+ уникальных адресов взаимодействовали с сетью за всё время
- 2M+ транзакций в день (исторический рекорд 3.6M+ 28 апреля 2026)
- $40B+ TVL в DeFi экосистеме (оценка DefiLlama)
- 5,000+ ежемесячных контрибьюторов по всем репозиториям экосистемы

**Позиция:** #2 по рыночной капитализации; базовый расчётный слой для DeFi, NFT, стейблкоинов и сети L2-решений.

## ТЕХНОЛОГИЯ

**Архитектура:**
- **Consensus layer:** Proof of Stake (The Merge, сентябрь 2022) — 1.1M+ активных валидаторов, finality ~4–6 сек
- **Execution layer:** EVM (Ethereum Virtual Machine) — индустриальный стандарт совместимости
- **L2 экосистема:** Arbitrum, Optimism, Base, zkSync, Polygon — масштабирование без ущерба безопасности L1
- **Sharding road:** EIP-4844 proto-danksharding (Dencun, март 2024) → full danksharding (roadmap)

**Ключевые инновации:**
- **EIP-1559** (авг. 2021): предсказуемые комиссии, 100% base fee сжигается — структурное дефляционное давление
- **The Merge** (сент. 2022): PoS, снижение эмиссии на ~90% vs PoW
- **Dencun/EIP-4844** (март 2024): blobs, снижение L2 fees в 10x+
- **Glamsterdam (предстоящий):** EIP-7928 (параллельная обработка транзакций), EIP-7732 (ePBS — on-chain building), цель 10,000 TPS

**Зрелость продукта:** Максимальная. 10+ лет непрерывного продакшн, крупнейшая battle-tested smart contract сеть.

**Аудиты безопасности:** ❌ Специфический список аудитов в данных не предоставлен. Ethereum имеет наиболее тщательно проверенную кодовую базу в индустрии (Prysm, Lighthouse, Teku, Nimbus на consensus; Geth, Erigon, Nethermind на execution). Нулевых L1-взломов за 10 лет.

## ДОХОД ПРОЕКТА И VALUE CAPTURE

### Наличие дохода

| Параметр | Значение |
|---|---|
| Есть ли доход | ✅ Да |
| Откуда | Priority fees (tips) → валидаторы; Base fees → сжигаются (burn) |
| Дополнительно | MEV (Maximal Extractable Value) → валидаторам/builders |
| Месячный run-rate | ⚠️ ~$100–300M (оценка; точные данные Token Terminal/DefiLlama без подписки недоступны) |
| Годовой run-rate (оценка) | ⚠️ ~$1.0–2.0B (высокая неопределённость) |
| Тренд | 🔽 Снижение (L2 абсорбируют активность с L1, снижая base fee) |
| Ожидание | 🔼 Glamsterdam → дешевле fees → больше объёма → рост revenue |
| Источник данных | Token Terminal, DefiLlama (подписка) |

### Влияние дохода на токен

| Механизм | Статус | Детали |
|---|---|---|
| Fee switch (доход → держателям) | ❌ | Нет прямого распределения ETH-holders |
| Buyback | ❌ | Нет программы buyback |
| Burn (% fees) | ✅ | 100% base fees сжигаются (EIP-1559) — косвенный value capture |
| Staking yield | ✅ | ~2.84–4.2% APY для 36M застейканных ETH |
| Treasury (EF) | ⚠️ | EF имеет значительный treasury (~$1B+); расходует на R&D/гранты |
| Governance only | ❌ | ETH не является governance-only токеном |

**Вывод:** ETH получает **косвенный** денежный поток через burn механизм (каждая транзакция уменьшает supply). Прямого buyback или dividend нет. Стейкеры получают реальный доход ~3–4% APY. Модель value capture ближе к «дефляционной commodity», чем к «дивидендному активу».
