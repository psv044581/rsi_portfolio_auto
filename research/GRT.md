## ЧТО ЭТО

**The Graph** — децентрализованный протокол индексирования и запросов к блокчейн-данным. Работает как «Google для блокчейна»: разработчики создают **subgraph'ы** (схемы индексирования), которые обрабатывают ончейн-события и предоставляют данные через **GraphQL API**.

- **Для кого:** Web3-разработчики, DeFi-протоколы, DApps
- **Ключевые пользователи:** Uniswap, Aave, Compound, ENS и тысячи других протоколов
- **Запущен:** Mainnet — декабрь 2020, TGE — 17 декабря 2020
- **Реальные пользователи:** ✅ Да — миллионы запросов ежедневно, интеграции в Ethereum, Arbitrum, Polygon, Near, Avalanche, Solana и др.
- **Нарратив 2026:** Переход к AI-инфраструктуре (x402, A2A гейтвеи для AI-агентов)

---

## ТЕХНОЛОГИЯ

**Архитектура:**
- **Indexers** — операторы нод, стейкают GRT, обрабатывают запросы, получают query fees + indexing rewards
- **Curators** — сигнализируют о ценных subgraph'ах, стейкая GRT (получают долю query fees)
- **Delegators** — делегируют GRT Indexers в обмен на долю вознаграждений
- **Subgraph** — открытая схема индексирования + GraphQL endpoint

**Ключевые инновации:**
| Технология | Описание |
|---|---|
| GraphQL над блокчейном | Стандарт де-факто в Web3-разработке |
| Decentralized Query Market | Рыночное ценообразование на запросы через GRT |
| **Substreams** | Высокопроизводительный стриминг блокчейн-данных (2023+) |
| **Horizon Protocol Upgrade** | Мультисервисная архитектура — запущена Dec 2025 |
| **Token API** | Производительный API для данных по токенам |

**Зрелость продукта:** 🟢 Production-ready. 5+ лет на рынке, активный GitHub (6 ключевых репозиториев в graphprotocol org).

**Аудиты безопасности:** Взломов не зафиксировано ✅. Проводились аудиты (ConsenSys Diligence). CertiK-скор: ❌ данных не предоставлено.
