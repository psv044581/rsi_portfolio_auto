## ЧТО ЭТО

**The Graph** — децентрализованный протокол индексации блокчейн-данных, позиционирующий себя как «Google для блокчейна». Позволяет разработчикам dApps делать GraphQL-запросы к исторически индексированным данным блокчейнов (Ethereum, Arbitrum, Base, Solana, Polygon и др.) через открытые **Subgraphs**.

- **Для кого:** Web3-разработчики, строящие dApps, DeFi-протоколы, NFT-платформы, AI-агенты (через GRC-20 Knowledge Graphs)
- **Запущен:** Mainnet — декабрь 2020; протокол основан в 2018 г.
- **Реальные пользователи:** 14,023 активных Subgraph, 200+ активных Indexers, >1.2 триллиона совокупных запросов, интеграции Uniswap, Aave, ENS и сотни других протоколов
- **Ключевые участники сети:** Indexers (стейкают GRT, обрабатывают запросы), Delegators (делегируют GRT Indexers), Curators (сигнализируют, какие Subgraphs индексировать)

## ТЕХНОЛОГИЯ

### Архитектура
| Компонент | Описание |
|---|---|
| **graph-node** | Rust-ноды, индексирующие события смарт-контрактов в реальном времени |
| **Subgraphs** | Открытые API-схемы, описывающие, какие данные индексировать (WASM mappings) |
| **Substreams** | Потоковый low-latency pipeline для real-time трансформации блокчейн-данных (StreamingFast) |
| **Firehose** | High-performance инструментарий для извлечения raw блок-данных |
| **GRC-20** | Новый стандарт Knowledge Graphs — AI-читаемые онтологии для LLM-агентов |
| **Arbitrum Settlement** | Протокол перешёл на Arbitrum для сниженных gas fees |

### Ключевые инновации
- **Horizon (Dec 2025)** — унифицированный фреймворк, объединяющий Subgraphs, Substreams и новые data services в единую архитектуру
- **GRC-20 Knowledge Graphs** — позиционирование как data layer для AI-агентов и LLMs
- **Sunrise of Decentralized Data (June 2024)** — полный переход всех Subgraphs на децентрализованную сеть Indexers (критический milestone)

### Зрелость продукта
- **Продакшн:** 5+ лет работы, trillion+ запросов — боевая система
- **Аудиты:** ❌ не найдено конкретных аудиторов в предоставленных данных; история без крупных взломов (5+ лет) косвенно подтверждает качество кода
- **GitHub:** graph-node (Rust, 3,133 commits), indexer (TypeScript, 255 commits) — активная разработка; 30-дневная статистика ❌ не найдена
