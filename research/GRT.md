## ЧТО ЭТО

**The Graph (GRT)** — децентрализованный протокол индексации и API для блокчейн-данных. Работает как «Google для блокчейна»: разработчики публикуют *subgraphs* (открытые API на GraphQL), которые обрабатывают операторы-индексеры. Запущен в декабре 2020 года, TGE на Coinbase и других биржах.

**Для кого:** разработчики dApps, нуждающиеся в структурированных on-chain данных без необходимости держать собственный полный узел. Ключевые ниши: DeFi-аналитика, NFT-маркетплейсы, GameFi, DAO-инструменты.

**Реальные пользователи:** 14 023 активных субграфа (ATH Q2 2025), 200+ активных индексеров, 1.2+ трлн суммарных запросов за всю историю. Крупные потребители данных: Uniswap, Aave, Compound, OpenSea, Lido, Synthetix и десятки других ведущих протоколов.

**Milestone 2024:** «Sunrise of Decentralized Data» (июнь 2024) — все субграфы переведены с централизованного hosted service на децентрализованную сеть индексеров. **Milestone 2025:** Horizon mainnet (дек 2025) — унифицированная архитектура для Subgraphs, Substreams и новых типов данных.

## ТЕХНОЛОГИЯ

**Архитектура:**
Протокол состоит из нескольких участников с экономическими стимулами:
- **Indexers** (операторы нод) — стейкают GRT, обрабатывают запросы, получают query fees + indexing rewards
- **Curators** — сигнализируют на качественные субграфы через механизм «curation signal» (bonding curve), получают 10% от query fees субграфа
- **Delegators** — делегируют GRT индексерам, получают часть вознаграждений
- **Developers** — публикуют субграфы и платят query fees в GRT

**Ключевые инновации:**
- **GraphQL API** поверх блокчейна — значительно снижает сложность разработки dApps
- **Substreams** — потоковая обработка данных с высокой пропускной способностью (StreamingFast, 2022)
- **Firehose** — прямая трансляция блочных данных для аналитики
- **GRC-20 (2024–2025)** — новый стандарт для «Knowledge Graphs», читаемых AI-агентами и LLM-системами
- **Horizon (дек 2025)** — унифицированный фреймворк, объединяющий Subgraphs, Substreams и новые data services в единую архитектуру

**Зрелость продукта:** Высокая. Протокол работает с 2020 года, обработал 1.2+ трлн запросов. graph-node (Rust) имеет 3 133 коммита, indexer (TypeScript) — 255 коммитов. Активная разработка подтверждена.

**Аудиты безопасности:** ❌ специфичных данных не предоставлено. Нет зафиксированных хаков или критических взломов за 5+ лет работы. Наличие аудитов предполагается, конкретные компании/даты не указаны в исходных материалах.
