## ЧТО ЭТО

**Render Network** — децентрализованная сеть GPU-вычислений, соединяющая:
- **Поставщиков** (node operators) — владельцев простаивающих GPU
- **Потребителей** — 3D-художников, студии, разработчиков AI/ML-моделей

**Запуск:** TGE — 6 октября 2020 г. (Ethereum, RNDR). Миграция на Solana — ноябрь 2023 г. (RENDER).

**Реальные пользователи:** Да. 74.4M фреймов обработано исторически, 22M+ за 2025 год. 5,600 активных нод-операторов. Клиенты — студии цифрового искусства, generative AI компании, Refik Anadol (знаменитый AI-художник).

**Материнская компания:** OTOY Inc. (2008) — лауреат премии Academy Award за технологии рендеринга.

---

## ТЕХНОЛОГИЯ

**Архитектура:**
- Peer-to-peer сеть GPU-нод, координируемых через смарт-контракты на Solana
- **Burn-Mint Equilibrium (BME):** Job-запрос оценивается в USD → сжигается эквивалент RENDER → нодам минтятся свежие RENDER из резерва как оплата
- **Dispersed** — распределённая subnet-сеть GPU (активно развивается, public beta 2026)
- **Render Workflows** — SDK для TypeScript/Python (public beta с апреля 2026)
- Интеграция Cinema 4D (плагин поддерживается), поддержка NVIDIA H100/H200/A100, AMD MI300 (pending RNP-021)

**Ключевые инновации:**
- Первопроходец в decentralized GPU rendering (с 2017 whitepaper)
- Wormhole-миграция Ethereum → Solana без потерь для держателей
- BME — дефляционная модель, автоматически связывающая спрос на вычисления с ценой токена
- Subnets (RNP-023): модульная архитектура для расширения типов GPU-задач (AI training, high-VRAM)

**Зрелость продукта:** 🟢 Производственная стадия. Продукт работает 6+ лет.

**Аудиты безопасности:** ✅ Хаков не зафиксировано за 2024–2026. Детальные аудит-репорты в данных не найдены.
