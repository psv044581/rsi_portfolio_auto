## ЧТО ЭТО

**Starknet** — permissionless децентрализованный L2 validity rollup для Ethereum, использующий криптографические протоколы STARK (Scalable Transparent ARguments of Knowledge) для масштабирования без компромисса по безопасности Ethereum.

- **Для кого:** разработчики dApps, DeFi-пользователи, ищущие дешёвые транзакции с гарантиями Ethereum-уровня; теперь также BTC holders через strkBTC
- **Запуск:** TGE — 20 февраля 2024; Mainnet сети — 2022 год; StarkWare основана в 2018
- **Реальные пользователи:** ⚠️ спорно — дискуссия о «8 DAU» в январе 2026 (различие методологий); 245K транзакций/день при 2,369 уникальных адресов говорит об автоматизации/ботах

---

## ТЕХНОЛОГИЯ

**Архитектура:**
- **Язык Cairo:** собственный язык программирования для создания STARK-совместимых смарт-контрактов; более выразительный, чем Solidity в ZK-контексте
- **Validity Rollup:** каждый батч транзакций сопровождается STARK proof, верифицируемым on-chain на Ethereum L1 — математически гарантирует корректность без доверия к sequencer
- **Dual-token consensus (сентябрь 2025):** уникальная модель — STRK (75%) + BTC (25%) для обеспечения безопасности сети

**Ключевые инновации:**
- Co-inventor STARK/FRI протоколов — Eli Ben-Sasson разработал базовую математику
- **Shinobi Privacy Upgrade (апрель 2026):** нативная in-protocol приватность для всех токенов + Bitcoin через ZK-proofs
- **STRK20 (март 2026):** native privacy features для ERC-20 стандарта
- **Post-quantum wallet S2morrow (апрель 2026):** Falcon-512 lattice-based подписи
- **strkBTC (май 2026):** Bitcoin в DeFi с redemption в native BTC

**Зрелость продукта:** Mainnet работает с 2022 г., регулярные обновления протокола. Grinta (сентябрь 2025) — децентрализованный sequencing, новый fee market.

**Аудиты безопасности:** ❌ не найдено в предоставленных данных (но для ZK rollup математическая корректность доказывается самим протоколом).
