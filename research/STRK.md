## ЧТО ЭТО

**StarkNet** — permissionless децентрализованный Layer 2 validity rollup на базе Ethereum, построенный StarkWare Industries. Использует криптографические протоколы STARK (Scalable Transparent ARgument of Knowledge) для масштабирования Ethereum без компромисса по безопасности.

- **Для кого:** DeFi-протоколы, разработчики dApps, институциональные пользователи, желающие низкие комиссии при безопасности Ethereum
- **Язык:** Cairo — собственный язык для STARK-доказуемых программ
- **Запущен:** Mainnet Alpha — 2021, токен STRK — 20 февраля 2024 (TGE + аирдроп)
- **Реальные пользователи:** ✅ Есть — DeFi, NFT, стейкинг, но объёмы упали вследствие EIP-4844

## ТЕХНОЛОГИЯ

**Архитектура:**
- **ZK-rollup (validity rollup):** транзакции батчуются off-chain, STARK-proof верифицируется on-chain в Ethereum
- **Cairo VM:** Turing-complete, STARK-native VM; Cairo 2.0 (2023) — значительный рефакторинг
- **Sequencer:** централизованный → переход к децентрализации через Grinta upgrade (сентябрь 2025)
- **Prover:** SHARP (Shared Prover) — объединяет доказательства нескольких транзакций
- **Grinta upgrade (сен 2025):** децентрализованный sequencing + новый fee market
- **Shinobi Privacy Upgrade (апр 2026):** нативная in-protocol приватность для всех токенов, включая BTC
- **STRK20 стандарт (мар 2026):** privacy-расширение для ERC-20-подобных токенов
- **S2morrow wallet (апр 2026):** post-quantum кошелёк на Falcon-512 (lattice signatures)

**Зрелость продукта:** Mainnet работает с 2021, Cairo зрелый язык, активная экосистема разработчиков

**Аудиты безопасности:**
- CertiK выявил уязвимость full node (ноябрь 2024) — исправлено
- Регулярные внутренние аудиты StarkWare
- Сетевой сбой 5 января 2026 (18 мин revert, 4 ч downtime) — отчёт опубликован
- Экосистемный exploit: zkLend $9.5M (февраль 2025) — протокол третьей стороны, не core
