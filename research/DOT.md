## ЧТО ЭТО

Polkadot — гетерогенный мульти-чейн протокол, разработанный для обеспечения межблокчейнового взаимодействия и общей безопасности. Проект создан Гэвином Вудом — со-основателем Ethereum и автором Solidity, Yellow Paper и EVM-спецификации.

- **Для кого:** Разработчики, создающие специализированные блокчейны (парачейны), которым нужна общая безопасность без самостоятельного bootstrapping validator set
- **Mainnet запущен:** 26 мая 2020 (фаза II); токен в открытой торговле с 17 июля 2020
- **ICO:** Октябрь 2017 ($145M, $0.29/DOT)
- **Реальные пользователи:** 65 активных парачейнов — KILT Protocol (500k+ верифицированных credentials в Германии и Швейцарии), Energy Web (2M+ сертификатов возобновляемой энергии), PEAQ (IoT/RWA, +500% QoQ транзакции Q3 2025), Moonbeam (21.4% месячных активных адресов)
- **Текущий статус:** Polkadot 2.0 (Elastic Scaling) внедрён в Q1 2025; переход к JAM (Join-Accumulate Machine) запланирован на конец 2026 – начало 2027

## ТЕХНОЛОГИЯ

**Архитектура:**
- **Relay Chain** — центральный слой консенсуса и безопасности (NPoS + BABE финализация блоков + GRANDPA финализация цепи)
- **Parachains** — специализированные блокчейны, арендующие вычислительные слоты и использующие shared security реле-чейна
- **XCM (Cross-Consensus Message Format)** — стандарт межчейнового обмена сообщениями и ценностью
- **Bridges** — соединение с внешними сетями (Ethereum, Bitcoin)
- **Elastic Scaling (Polkadot 2.0, Q1 2025)** — динамическое выделение ресурсов парачейнам, 3–5x прирост throughput без изменения базового протокола

**Ключевые инновации:**
- **Heterogeneous sharding** — первая production-ready реализация разнородного шардинга в индустрии
- **Shared security model** — парачейны получают полную безопасность реле-чейна с первого блока
- **NPoS (Nominated Proof-of-Stake)** — более равномерное распределение stake по валидаторам vs DPoS
- **OpenGov** — полностью on-chain governance без постоянного совета, с референдум-треками разной срочности

**JAM (Join-Accumulate Machine) — следующее поколение (2026–2027):**
- Замена relay chain на "blockchain supercomputer"
- Объединяет compute + storage в единую среду выполнения (CoreVM)
- Устраняет концепцию фиксированных parachain-слотов в пользу flexible coretime
- Потенциально кратно расширяет возможности экосистемы по сравнению с Polkadot 1.0

**Зрелость продукта:** Production-ready. Elastic Scaling live. 65 активных парачейнов. 5 лет mainnet.

**Аудиты безопасности:** ✅ Множественные аудиты (Trail of Bits, SRLabs и др.), проводятся регулярно. ❌ Конкретные отчёты за 2025–2026 в предоставленных данных отсутствуют. Нет зафиксированных крупных security инцидентов.
