## ЧТО ЭТО

**Chia Network** — Layer 1 блокчейн с инновационным консенсусом **Proof of Space and Time (PoST)** вместо Proof of Work. Основан Брамом Коэном (создатель BitTorrent) в 2017 году. Mainnet запущен **19 марта 2021 года**.

Вместо видеокарт и ASIC-майнеров фармеры используют **свободное место на жёстких дисках** — идеология «зелёного» блокчейна. Для смарт-контрактов создан собственный язык **Chialisp** (на основе Lisp).

**Реальные пользователи:**
- Корпоративный сегмент: Climate Action Data Trust (Всемирный банк, IETA, правительство Сингапура)
- Институциональный: Chia Vaults (multi-sig custody), Permuto Capital (токенизация акций США)
- Потребительский: Chia Cloud Wallet (released Apr 2026, ACH покупки в США)

Проект ориентирован на **RWA (реальные активы)** и **корпоративные применения**, а не DeFi-ретейл.

## ТЕХНОЛОГИЯ

| Компонент | Описание |
|---|---|
| **Консенсус** | Proof of Space and Time — первый новый Nakamoto-стиль с 2008 года |
| **PoSpace** | Фармеры выделяют место на диске, seed заполняется cryptographic plots |
| **PoTime** | VDF (Verifiable Delay Function) — временная задержка исключает pre-farming атаки |
| **Chialisp** | Smart transaction language — функциональный, на базе Lisp, фокус на аудируемости |
| **DataLayer** | Децентрализованное хранилище данных с on-chain Merkle-деревьями (основа CADT) |
| **CAT (Chia Asset Tokens)** | Стандарт fungible токенов (аналог ERC-20); была уязвимость CAT1 → переход на CAT2 в 2022 |
| **Chia Vaults** | m-of-n multi-sig кастодиальное решение (запущено янв. 2025) |

**Зрелость продукта:** Mainnet 5+ лет, v2.7.x, активный GitHub (#18 по dev-активности), 3,475 коммитов, 11k stars.

**Аудиты безопасности:** В 2023 была устранена уязвимость Offer File (CAT1 → CAT2 миграция, блок 2,311,760). В мае 2026 выпущен v2.7.1 с **критическими исправлениями безопасности** после CVE — требовалось экстренное обновление.

**Soft fork (блок 8,655,000, апрель 2026):** Новые rate limits — технически значимое обязательное обновление.
**Chia 3.0 (hard fork):** Полная переработка consensus cost — дата не объявлена (ключевой технологический риск).
