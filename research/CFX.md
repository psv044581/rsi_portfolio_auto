## ЧТО ЭТО

Conflux (CFX) — публичный блокчейн Layer 1, запущенный в октябре 2020 года. Использует оригинальный консенсусный механизм **Tree-Graph** (разновидность DAG-блокчейна) с гибридным PoW/PoS (переход завершён в 2022). Ключевое позиционирование: **единственный регуляторно-совместимый публичный блокчейн в Китае**, что даёт уникальное преимущество для проектов, нацеленных на азиатский рынок.

**Архитектура dual-space:**
- **Core Space** — нативный, транзакции в CFX, Tree-Graph VM
- **eSpace** — EVM-совместимое пространство: Solidity, MetaMask, портирование ETH-проектов без изменений

**Для кого:** DeFi-протоколы, стейблкоины, NFT-платформы, AI/Web3 проекты — преимущественно азиатские. Cross-chain bridge кумулятивно обработал $800M+.

**Реальные пользователи:** Интеграции с Tether (USDT0, CNHT0), Tether Gold (XAUt0), LayerZero OFT; TVL ~$120M (начало 2026). Активность скромная для 6-летней сети — $0 протокольных fees говорят о низком органическом использовании.

## ТЕХНОЛОГИЯ

**Ключевые инновации:**

| Компонент | Описание |
|---|---|
| **Tree-Graph Consensus** | DAG-структура вместо линейного блокчейна. Боковые блоки включаются в граф (не отбрасываются как uncle в ETH), параллельная обработка транзакций. Разработан Тьюринговским лауреатом А. Яо |
| **Гибридный PoW/PoS** | PoW генерирует блоки, PoS-комитет из 1000 нод финализирует чекпоинты — защита от 51% атак |
| **Dual Space** | Core Space (нативный Conflux VM) + eSpace (EVM-compatible, Solidity/MetaMask) |
| **15,000 TPS** | Достигнуто после hardfork v3.0.1 (август 2025) |
| **AI Smart Contracts** | Объявлены в v3.0.1 — интеграция AI-логики прямо в смарт-контракты (детали не раскрыты) |
| **CIP-166** | Count Leading Zeros opcode (v3.0.3) — расширение EVM-инструментария |

**Зрелость продукта:**
- Mainnet работает с октября 2020 (6 лет непрерывно)
- Собственная инфраструктура: Fluent Wallet, Sirius block explorer, Confura RPC
- 120+ GitHub репозиториев, активные релизы в 2026

**Аудиты безопасности:** ❌ Конкретные данные о сторонних аудитах в предоставленных материалах **не найдены**. Код опубликован под GPL-3.0 (conflux-rust, 719 ⭐). Сеть работает 6 лет без зафиксированных критических взломов.
