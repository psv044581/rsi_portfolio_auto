## ТЕХНОЛОГИЯ

### Архитектура: Multi-Role Consensus
Flow решает blockchain trilemma через разделение ролей нод на четыре типа:
1. **Collection Nodes** — батчинг транзакций
2. **Consensus Nodes** — финализация порядка транзакций (PoS)
3. **Execution Nodes** — выполнение вычислений
4. **Verification Nodes** — проверка корректности

Это позволяет масштабироваться вертикально без шардинга, сохраняя атомарность состояния.

### Ключевые инновации
| Инновация | Описание |
|-----------|----------|
| Cadence | Resource-oriented programming language; assets как first-class citizens; снижает риск double-spend |
| EVM Equivalence | Достигнута Dec 2024 (Crescendo upgrade); Ethereum-контракты работают нативно |
| Human-readable transactions | Пользователь видит что подписывает, а не hex-данные |
| Account abstraction | Нативная, без отдельного стандарта типа ERC-4337 |

### Зрелость продукта
- Mainnet работает с 2020 года — 5+ лет без крупных остановок (до Dec 2025)
- ~60,000 смарт-контрактов задеплоено после Crescendo upgrade
- #1 L1 по количеству проектов на ETHGlobal 2025

### Аудиты безопасности
- ⚠️ **Dec 2025 exploit**: создано ~$3.9M фальшивых FLOW через уязвимость протокола — критический прецедент
- Governance committee смогла изолировать активы on-chain — показатель централизованного контроля
- Rollback план отвергнут после community backlash — позитивный сигнал для децентрализации
- Специфические аудиторы (CertiK и пр.) в предоставленных данных: ❌ не найдено
