## ЧТО ЭТО

**Flow** — Layer 1 блокчейн, разработанный **Dapper Labs** специально для потребительских приложений и цифровых активов (NFT, игры, спортивные коллекционные карточки). Запущен в mainnet 27 января 2021 года. Архитектура оптимизирована для высокой пропускной способности без шардинга. Работает на Proof of Stake с многоролевой архитектурой (Collector, Consensus, Execution, Verification nodes).

**Реальные пользователи существуют:**
- NBA Top Shot (первый мейнстримный NFT-продукт)
- NFL All Day — 600,000+ зарегистрированных пользователей
- LaLiga Golazos — 400,000+ пользователей (Европа + Латинская Америка)
- Disney Pinnacle — legendary editions распродаются мгновенно
- 24Karat Japan — 250,000 еженедельных пользователей через вендинг-машины

Сеть имеет реальную аудиторию, однако активность сосредоточена в нишевом NFT-сегменте, а не в DeFi.

## ТЕХНОЛОГИЯ

**Архитектура:**
- **Multi-role (pipeline) consensus** — задачи сети разделены между 4 типами нод (Collector, Consensus, Execution, Verification). Позволяет масштабировать без шардинга.
- **Cadence** — оригинальный язык смарт-контрактов с resource-oriented programming. Версия 1.0 (post-Crescendo upgrade) с entitlements и view functions. Ориентирован на безопасность и удобство разработчиков.
- **EVM Equivalence** — полная совместимость с Ethereum Virtual Machine (добавлена в 2024), позволяет портировать Solidity-приложения.
- **~60,000 новых смарт-контрактов** развёрнуто со времени Crescendo upgrade (2025–2026).
- **#1 most-built-on L1 на EthGlobal hackathons (2025)** — 1 из 4 участников выбрал Flow.

**Cross-chain интеграции:**
- Stargate (LayerZero) — 100+ блокчейнов
- Axelar Interchain Amplifier — 70+ блокчейнов
- PayPal PYUSD стейблкоин живёт на Flow

**Зрелость продукта:** Высокая для NFT-вертикали, низкая для DeFi ($12.94M TVL).

**Аудиты безопасности:** ⚠️ **Критический инцидент:** Декабрь 2025 — infinite mint vulnerability в Cadence runtime позволила создать ~87.4 миллиарда поддельных FLOW токенов. ~$3.9M выведено через Celer, DeBridge, Relay, Stargate. Уязвимость устранена, поддельные токены уничтожены (30 янв. 2026), 50.34M реальных FLOW сожжено в рамках buyback (23 фев. 2026). Это серьёзнейший security incident для L1.
