## ЧТО ЭТО

**Worldcoin (WLD)** — протокол глобальной цифровой идентификации и финансовой сети, построенный на принципе **Proof of Personhood** (доказательство человечности). Ключевая инновация — **World ID**: приватный криптографический идентификатор, получаемый после сканирования радужной оболочки глаза устройством **Orb** (собственная разработка).

- **Для кого:** Для всех, кто хочет доказать, что они — живой человек в интернете (защита от ботов, AI-агентов, дипфейков). Целевые рынки: DeFi, игры, социальные сети, корпоративные сервисы.
- **Когда запущен:** Tools for Humanity основана в октябре 2019; первый листинг токена **~июль 2023**; World Chain (L2 на OP Stack) — 2024.
- **Реальные пользователи:** 38M+ пользователей World App, ~18M верификаций через Orb в 160 странах. Партнёрства: Visa (дебетовая карта), Match Group (Tinder), Razer, AgentKit (Coinbase). Однако **on-chain активность критически низкая** — только 117K активных кошельков.

## ТЕХНОЛОГИЯ

**Архитектура:**
- **Orb** — специализированное устройство (iris scanner + мультиспектральная камера), собирает биометрические данные для создания **IrisHash** — уникального числового отпечатка, не позволяющего восстановить оригинальное изображение
- **World ID** — ZK-proof идентификатор (zero-knowledge proof) на базе Semaphore protocol; пользователь доказывает уникальность без раскрытия биометрии
- **World Chain** — L2 Ethereum на OP Stack (Optimism); транзакции верифицированных пользователей приоритизированы и субсидированы
- **World App** — мобильное приложение (custody кошелёк + интерфейс для World ID)

**Ключевые инновации:**
1. Единственный протокол с **физическим hardware verification** в крипто-пространстве — невозможно подделать без присутствия человека
2. **ZK privacy** — биометрия не хранится на цепочке, только хэш для доказательства уникальности
3. **Human-priority transactions** на собственном L2 — новая парадигма gas fee для верифицированных людей

**Зрелость продукта:** Продукт в production с реальными пользователями. World Chain работает. Orb устройства развёрнуты в 160 странах.

**Аудиты безопасности:** ❌ Конкретные данные по аудиторам и датам не предоставлены. Проект open-source (orb-software, ampc-common, world-chain на GitHub).

**Главный риск технологии:** Централизация через Orb — все пользователи зависят от физической инфраструктуры Tools for Humanity.

При $500M revenue → FDV ~$15B → цена ~$1.50, то есть **+255%** (3.5x)

### 🔴 Bear Case (−70–90%)
